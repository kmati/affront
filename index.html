Welcome to Affront

Affront was made to be simple to use and small in size. It offers you the typical virtual DOM, template-based component, client-routing framework stuff you're used to. Affront does not aim to be overly opinionated on how you should code the front-end.

Why Affront?

Affront was created to be a minimalist framework. It allows you to implement the front-end using the features we love, entirely within JavaScript, HTML templates and styling.
